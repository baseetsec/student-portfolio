    How This Works

    The reasoning behind the tool

    This dashboard simulates the shape of a real vulnerability scan: a target is entered, a scan sequence runs, and a findings report is produced mapping issues to real OWASP Top 10 categories with severity ratings and remediation guidance. Rather than random results, the report is generated deterministically from the text you enter the same input always produces the same findings, the way a real scanner would behave consistently against the same target. Building an actual scanner that sends live requests to arbitrary sites is well outside the scope of a safe, honest student project, so this focuses on what matters for learning: correctly identifying and explaining real security categories.
